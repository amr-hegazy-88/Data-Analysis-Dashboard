# Food-Sales-Mart-Data-Analysis-project(interactive dashboard creation using Power Bi , power query , DAX )
 ## Project Objective
The objective of this project is to perform a comprehensive data analysis of "Food Mart" sales for 1997,1998 using Power BI to uncover actionable insights that drive strategic business decisions. The project focuses on:
Analyzing sales performance and identifying revenue trends over time.
Identifying top-selling products and high-margin categories.
Monitoring inventory levels and evaluating supply chain efficiency.
Analyzing customer demographics and geographic sales distribution to optimize marketing efforts.
Developing interactive dashboards to track Key Performance Indicators (KPIs) effectively.

# Key Performance Indicators (KPIs)
- The dashboard tracks several critical metrics:
- Financials: Total Revenue, Gross Profit, and Profit Margin %.
- Productivity: Top 5 Best-Selling Products and Category Performance.
- Operations: Total Units Sold and Average Transaction Value.
- Customers: Total Unique Customer Count and Sales by Region/City.
# Tech Stack
- Power BI Desktop: For report development and data visualization.
- Power Query: Used for the ETL (Extract, Transform, Load) process to clean and shape the data.
- DAX (Data Analysis Expressions): Implemented for advanced calculations and custom business measures.
- Data Modeling: Established a Star Schema to ensure efficient data processing and accurate reporting.
# Key Insights from the Analysis
- Top Categories: Identified which departments (e.g., Produce, Snack Foods, Beverages) contribute most to the bottom line.
- Geographic Leaders: Discovered which store locations are over-performing in terms of profit per square foot.
- Customer Trends: Analyzed how different customer segments (based on income or education) impact sales volume.

# The Process
- I followed a structured data analysis workflow to ensure accuracy and clarity:
- Data Connection: Integrated multiple data sources related to Food Mart sales into Power BI.
- ETL & Cleaning (Power Query): Cleaned raw data by removing duplicates, handling missing values, and transforming columns to ensure consistent data types.
- Data Modeling: Designed a Star Schema with a centralized Sales Fact table connected to Dimension tables (Store, Product, Customer, and Date) for optimized performance.
- DAX Development: Authored custom DAX measures to calculate essential metrics like Total Revenue, Net Profit, and Growth Rates.
- Dashboard Design: Created an interactive and user-friendly interface using specialized visuals, slicers, and drill-through features to provide deep-dive insights.
